## Featured Projects

### Enhancing Privacy in Location-Based Services Using a Hybrid AOA-TDOA Positioning Technique (Final Year Project) | iNOTEK Series I 2026 Silver Award
<p align="justify">
This project develops a privacy-aware localization framework for Location-Based Services by combining the Angle of Arrival and Time Difference of Arrival methods. Simulations in MATLAB demonstrate improved positioning accuracy while enabling a proximity-based access mechanism to protect sensitive location information.
</p>

### Automated Water Quality Monitoring System for Pipelines (Integrated Design Project) | iNOTEK Series 1 2025 Silver Award
<p align="justify">
This project implements a sensor-based monitoring system to measure key water quality parameters, including pH, turbidity, and temperature in pipeline environments. The collected data is processed by a microcontroller and displayed in real time on an LCD interface, with LEDs serving as indicators for immediate user monitoring.

### Poomsae Marking System
<p align="justify">
This project implements a microcontroller-based scoring system for Taekwondo Poomsae competitions, where judges input scores through push buttons connected to embedded devices. ## Coursework Group Project
### Auto Lighting System for Street Lights
<p align="justify">
This project develops an automated street lighting system using an LDR sensor and the STM32F446RE to detect ambient light and control lighting without manual intervention. Real-time data is transmitted via the ESP8266 ESP-01 to the ThingSpeak dashboard for remote monitoring.
</p>
<p align="justify">
LEDs are used to simulate street lights in the prototype, demonstrating a practical integration of embedded systems and IoT for energy-efficient smart city applications.
</p>
